## 🚀 Fluxo de Trabalho

### Backend
1. Ler [`BACKEND.md`](BACKEND.md)
2. Configurar ambiente (instalar Node, npm, criar .env)
3. Ler documentação do seu módulo
4. Implementar (Controller → Service → Repository)
5. Testar no Postman
6. Fazer Pull Request

### Frontend
1. Ler [`FRONTEND.md`](FRONTEND.md)
2. Configurar servidor local (Live Server ou `npx serve`)
3. Ler documentação da sua tarefa
4. Implementar módulos
5. Testar no navegador
6. Fazer Pull Request
